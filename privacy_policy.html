Privacy Policy

Last updated: March 2026

Mirra ("the Extension") is committed to protecting your privacy. This policy explains how the Extension handles your data.

Data Collection

Mirra does not collect, transmit, or share any personal data. All data you create within the Extension (pinned text fragments, notes, API keys, and generated insights) is stored exclusively in your browser's local storage (chrome.storage.local) on your own device.

Third-Party API Keys

If you choose to enter an API key (e.g., Gemini, DeepSeek, or OpenAI), that key is stored only in your browser's local storage and is used solely to make requests directly from your browser to the respective AI provider. Your API key is never sent to any Mirra server — because no such server exists.

Host Permissions

The Extension requests access to aistudio.google.com solely to enable text selection and pinning functionality within Google AI Studio. No data from that site is collected or transmitted by the Extension.

Data Sharing

We do not sell, share, or transfer any user data to third parties.

Contact

If you have questions about this privacy policy, please contact the developer through the Edge Add-ons store listing.
