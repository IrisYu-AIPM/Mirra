# ✦ Mirra — Your Thinking Mirror

> **Your thinking has a shape. Mirra shows you what it is.**
>
> 你每次跟 AI 的深度碰撞，都在塑造一个思维的你。Mirra 把它照给你看。

---

## 这是什么

Mirra 是一个 Chrome 插件，专为 Google AI Studio 设计。

它让你在对话时**精准划词钉选**有价值的片段，然后把这些片段蒸馏成属于你自己的认知资产——包括你真正在困扰什么、你的信念发生了什么转变、你接下来应该做什么，以及下一场对话的开场白。

越积累，Mirra 越懂你。随着时间推移，你会在资产库里看见一幅持续进化的**认知画像**。

---

## 安装方法（需要 1 分钟）

1. 解压这个 zip 文件到任意本地文件夹
2. 打开 Chrome，地址栏输入 `chrome://extensions`
3. 右上角开启 **「开发者模式」**
4. 点击 **「加载已解压的扩展程序」**，选择刚才解压的文件夹
5. 安装完成，工具栏出现 Mirra 图标

---

## 使用方法

### 钉选片段
1. 打开 [Google AI Studio](https://aistudio.google.com)
2. 在对话中**选中任意文字**，出现浮动工具条
3. 点 **📌 钉选** 直接保存，或点 **✍️ 批注** 加上你的想法后保存
4. 钉选的文字会在页面上高亮显示

### 生成 Mirra 洞察
1. 钉选若干片段后，点击右侧面板底部的 **「✦ 生成 Mirra 洞察」**
2. 需要提前在「API 配置」里填入 API Key（支持 Gemini / DeepSeek / OpenAI）
3. 生成后会自动保存到 **Mirra 资产库**

### 查看资产库
- 生成完成后点击 **「查看全部 →」**，在新标签页打开资产库
- 每条记录可以**展开全部**查看所有内容，包括你钉选的原文
- 积累 3 次以上后，顶部会出现**认知画像**区块
