# ✦ Mirra — Your Thinking Mirror

> **你不知道的，AI 都知道。**
>
> 你和 AI 聊了几万个字——但你只用到了它能力的 20%。真正有价值的那个象限，是「AI知道、你不知道」。Mirra 帮你拿回那 80%。

## 这是什么

Mirra 是一个 浏览器 插件，专为 Google AI Studio 设计。

它让你在对话时**随手钉选**有价值的片段，配合实时随记，然后把这些内容蒸馏成属于你自己的认知资产——包括你真正在困扰什么、被撼动的信念是什么、本次的思维偏向，以及下一步该怎么走。

越积累，Mirra 越懂你。积累 2 次以上对话后，解锁**命运镜像**——从所有历史数据中提炼你的 认知 DNA，推演你的两条平行轨迹，并直接指出那个你一直没看见的核心盲点。

---

## 安装方法（需要 1 分钟）

1. 解压这个 zip 文件到任意本地文件夹
2. 打开 Chrome/edge等浏览器，地址栏输入 `chrome://extensions`、`edge://extensions/`
3. 右上角开启 **「开发者模式」**
4. 点击 **「加载已解压的扩展程序」**，选择刚才解压的文件夹
5. 安装完成，工具栏出现 Mirra 图标

## 使用方法

### 钉选片段
1. 打开 [Google AI Studio](https://aistudio.google.com)
2. 在对话中**选中任意文字**，出现浮动「📎 钉选」按钮
3. 点 **📌 钉选** 直接保存，片段实时同步到侧边栏，或点 **✍️ 批注** 加上你的想法后保存
-金色：你说的话
-蓝色：AI 说的话
-绿色：你的随手笔记
4. 对话中随时在「实时随记」栏写下脑中冒出的联想、疑问和灵感，会和钉选片段一起进入分析

### 生成 Mirra 洞察
1. 钉选若干片段后，点击右侧面板底部的 **「✦ 生成 Mirra 洞察」**
2. 需要提前在「API 配置」里填入 API Key（支持 Gemini / DeepSeek / OpenAI）
3. 生成后会自动保存到 **Mirra 资产库**
4. 生成内容包括：
🎯 你真正困扰的是什么（表面问题 vs 实际困扰 vs 这次看见了）
⚡ 这次被撼动的信念（旧信念 → 新认知）
📊 本次认知偏向（执行型 / 战略型 / 情绪型 / 分析型）及持续预警

### 命运镜像（2 次对话后解锁）
积累 2 次以上对话记录后，资产库顶部解锁命运镜像 · The Mirror of Destiny：
Cognitive DNA：从你说过的话中提炼有证据支撑的思维特征
2035 年双轨推演：基于你的模式，推演两条平行的未来轨迹
核心盲点：直接指出你一直没看见的那个东西——基于你自己的话，无法反驳

### 查看资产库
- 生成完成后点击 **「查看全部 →」**，在新标签页打开资产库
- 每条记录可以**展开全部**查看所有内容，包括你钉选的原文

## 核心理念

信息是消耗品，资产是增值的。

大多数人用 AI 只停留在「查资料、写代码、总结翻译」——这是效率工具，但不改变认知。Mirra 的目标是带你进入真正的甜蜜区：让 AI 帮你看见你自己从来看不见的东西。

每次深度对话都留下痕迹，随时间复利增长。
